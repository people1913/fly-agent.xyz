Trust Record
信任记录
这是什么
Trust Record 是 Fly 的不可篡改信任档案。它累积所有 Attestation 证明，为每个 Agent 生成可查询、可审计的信任记录。
为什么需要
单次验证通过了——但这个 Agent 长期可信吗？
没有 Trust Record，你只能看到「这一次是真的」。但商业合作需要长期信任，需要知道「这个 Agent 一直可靠」。
怎么工作
STEP 1
证明累积
所有 Attestation 证明自动写入 Trust Record，不可篡改
STEP 2
信任聚合
基于历史记录，计算 Agent 的长期可信度和声誉
STEP 3
可查询档案
任何人都可以查询 Agent 的信任记录，支撑商业决策
下一步
Trust Record 证明了「这个 Agent 可信」，接下来需要转化为可结算的价值——这就是 Settlement Ready。
