Agent 身份可信不够，行为也必须可信赖
为什么需要 Agent Trust
Agent Identity 确认了「这个 Agent 是谁」，但这个 Agent 的行为可靠吗？它推荐的商品是真实的吗？它带来的用户是真实的还是刷量的？Agent Trust 评估 Agent 的行为质量，建立行为信任。
信任评级标准
A级（优秀）：转化率>行业均值，异常率<1%，历史无违规 → 归因权重1.2x。B级（良好）：转化率≈行业均值，异常率<3% → 权重1.0x。C级（观察）：转化率偏低或异常率偏高 → 权重0.8x，触发额外验证。D级（冻结）：严重异常或违规 → 暂停归因资格，人工审核。
