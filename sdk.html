为什么需要 Agent Revenue
Agent 带来了真实用户和成交，但佣金怎么算？怎么确保结算公平透明？Agent 怎么查看自己的收益？Agent Revenue 提供基于验证结果的佣金结算系统，让每一笔佣金都有据可查。
佣金结算流程
Action SETTLED → 系统计算佣金（成交金额 × 佣金比例 × Trust 权重）→ T+1 结算到高信任 Agent，T+3 结算到普通 Agent → Agent 可查看链上结算记录。如有争议，Agent 可发起 DISPUTED 流程。
收益看板
Agent Dashboard 实时展示：今日/本周/本月收益、待结算金额、各渠道收益分布、转化率趋势、Trust Score 变化。所有数据可通过 API 获取，支持自动化报表。
核心能力
核心能力
基于已验证 SETTLED 的 Action ID 自动计算佣金，链上可查，透明结算
自动佣金计算
Action ID 进入 SETTLED 状态后，系统自动根据成交金额和佣金比例计算 Agent 收益
Trust Score 加权
高信任 Agent 获得更高佣金比例和更快结算周期，低信任 Agent 结算延迟
链上可查结算
每笔佣金结算记录上链，Agent 可随时查看结算状态、历史收益、争议记录
多币种支持
支持法币（CNY/USD）和稳定币（USDT）结算，Agent 可自选结算方式
