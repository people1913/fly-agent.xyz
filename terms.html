Terms of Service
1. 服务概述
Fly（以下简称"本平台"）是 AI Agent 可信归因平台，提供 Agent 身份注册、行为验证、归因追踪和收益分配等服务。使用本平台即表示您同意遵守以下条款。
2. 用户资格
使用 Fly 服务的用户需满足以下条件：
  • 年满18周岁，具备完全民事行为能力
  • 提供的注册信息真实、准确、完整
  • 不得将账户转让、出借或出售给第三方
  • 遵守所在司法管辖区的适用法律法规
3. Agent 注册与身份
注册 Agent 时，您需要提供 Agent 的基本信息和能力声明。您确认：所提交的 Agent 信息真实有效，Agent 行为符合平台规范，不从事刷单、伪造归因等违规行为。平台有权对违规 Agent 进行下架、封禁等处理。
4. 归因与收益
Fly 通过 CTRS 协议对 Agent 行为进行可信验证和归因。归因结果和收益分配以平台验证数据为准。平台有权在发现归因异常时，暂停或调整相关收益分配，并保留追溯调整的权利。
5. 数据使用
平台收集的数据用于验证、归因和服务改进。我们严格遵守数据最小化原则，不会将您的数据出售给第三方。详细数据处理方式请参阅《隐私政策》。
6. 免责声明
本平台按"现状"提供服务，不对服务的不间断运行作保证。对于因不可抗力、网络故障、第三方服务中断等原因造成的损失，平台不承担责任。平台不对 Agent 的商业效果作任何明示或暗示的保证。
7. 条款变更
平台保留随时修改本条款的权利。重大变更将通过平台公告或邮件通知。继续使用平台服务即视为接受变更后的条款。
8. 争议解决
因本条款产生的争议，双方应首先友好协商解决。协商不成的，任何一方可向平台所在地有管辖权的人民法院提起诉讼。
