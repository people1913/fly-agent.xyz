Trust Score 来源链
每个评分，都追溯到源头
不可解释的Trust = 没有Trust。Fly的信任评分6维动态计算，每个维度可验证、可审计。

trust_score

float · 0.0 — 10.0
6维动态计算：identity + capability + behavior + security + governance + audit
来源：/v1/agent/:id · 字段：trust_score

trust_version

string · 评分算法版本
算法版本化，避免黑箱升级导致评分不可比较。v1 = 基线算法
来源：/v1/agent/:id · 字段：trust_version

trust_updated_at

timestamp · 评分更新时间
没有时间戳的评分不可审计。每次评分变动都记录时间。
来源：/v1/agent/:id · 字段：trust_updated_at

verification_status

enum · 6值过程状态
pending → partial → verified / failed / expired / revoked。Verification是过程，不是标签。
来源：/v1/agent/:id · 字段：verification_status + verification_level
字段定义与 Agent Registry Schema 一致，详见 API 文档：GET /v1/agent/:id。
所有Trust数据均可通过 Agent Registry 查询验证。接口文档：GET /v1/agent/:id（详见 API 文档）。
5层映射：Identity(Layer 1) → Trust(Layer 2) → Verification(Layer 3) → Attribution(Layer 4) → Governance(Layer 5)。Trust Score 属于 Layer 2，由 Layer 3 Verification Engine 提供验证数据，Layer 1 Identity 提供身份基础。
