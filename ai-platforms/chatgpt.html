为什么需要 ChatGPT 归因验证
ChatGPT 是最大的 AI 对话平台，用户经常问「推荐一个 XX」。当用户通过 ChatGPT 推荐完成消费，商家如何证明这笔成交来自 ChatGPT 推荐？如何与 ChatGPT 生态的 Agent 开发者分润？
接入流程
① 注册 Fly 账号，创建 ChatGPT Agent → ② 集成 Fly SDK（npm install @fly/sdk），3行代码接入 → ③ 配置推荐链接模板和佣金比例 → ④ 测试验证（沙盒环境）→ ⑤ 上线生产

接入方案

推荐链接追踪
Agent 在 ChatGPT 回复中嵌入带 Behavior ID 的推荐链接，用户点击即产生归因记录
对话归因识别
区分 AI 主动推荐 vs 用户追问 vs 随机提及，只对真实推荐行为产生归因
防刷验证
识别模拟对话、批量点击、自动化推荐等作弊行为，保障归因数据真实
多 Agent 归因
同一用户可能收到多个 ChatGPT Agent 推荐，按多触点归因规则分配权重
