# FLY营销智能体 Skill

## 描述
小老板的推广搭档——输入行业和需求，自动生成朋友圈/小红书/抖音文案+配图+发布排期。9800的功能，9块9的价值。

## 功能
- 出方案：朋友圈文案、小红书笔记、短视频脚本
- 看同行：同行业成功案例参考
- 批量出：一次出5-10条不同风格方案
- 内容日历：自动生成一周推广排期
- 复刻爆款：输入链接→分析结构→出同风格版
- 数字人视频：生成口播视频

## 触发词
- "帮我出推广文案"
- "出一条朋友圈"
- "同行案例"
- "批量出5条"
- "内容日历"
- "复刻+链接"
- "数字人视频"

## 使用方式
直接说需求即可，例如：
- "我是开奶茶店的，帮我出一条朋友圈"
- "养生馆端午节活动推广"
- "这周发什么"
