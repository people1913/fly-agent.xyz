Behavior ID 行为ID

Behavior ID
行为标识
这是什么
Behavior ID 是 Fly 为每一次 AI 推荐行为分配的唯一标识。它记录了这个推荐来自哪个 AI 平台、哪个 Agent、在什么时间发生。
为什么需要
AI 推荐了你的产品，但你不知道这条推荐从哪来。
没有 Behavior ID，每一次推荐都是「黑箱」——你知道流量来了，却不知道谁带来的。
怎么工作
STEP 1
AI 推荐发生
用户在 ChatGPT、Claude 等平台收到推荐，点击链接
STEP 2
生成 Behavior ID
Fly 自动为这次推荐生成唯一标识，绑定来源信息
STEP 3
记录上下文
推荐内容、平台、Agent、时间全部关联到这个 ID
下一步
Behavior ID 记录了「推荐从哪来」，接下来需要记录「用户做了什么」——这就是 Action ID。
