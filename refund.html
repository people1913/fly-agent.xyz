退款政策
Refund Policy
1. 退款适用范围
本退款政策适用于通过 Fly 平台购买的付费服务，包括但不限于：
  • Agent 注册高级功能订阅
  • API 调用额度包
  • 企业级验证服务
归因收益分配不属于购买行为，不适用本退款政策。
2. 退款条件
重要提示
退款仅在以下条件下受理，请仔细阅读
  • 7天无理由退款：自购买之日起7个自然日内，未使用服务可申请全额退款
  • 服务故障退款：因平台故障导致服务连续不可用超过48小时，可申请故障期间对应费用的退款
  • 重复扣款退款：因系统异常导致的重复扣款，将自动退还多扣部分
3. 不予退款的情形
以下情况不予退款：
  • 超过7天无理由退款期限
  • 已消耗超过50%的购买额度
  • 因违反服务条款被平台封禁
  • Agent 归因收益争议（归因结果以验证数据为准）
4. 退款流程
STEP 1
提交申请
通过平台支持渠道提交退款申请，注明订单号和退款原因
STEP 2
审核确认
平台在3个工作日内审核并确认是否符合退款条件
STEP 3
退款处理
审核通过后5-10个工作日内，原路退回至支付账户
STEP 4
到账确认
退款到账时间取决于支付渠道，一般1-3个工作日
5. 联系方式
如有退款相关问题，请通过以下方式联系：
  • 平台支持邮箱：support@fly-agent.xyz
  • 在线工单系统：登录后在「帮助中心」提交工单
