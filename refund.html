Fly 退款政策：保障客户权益，明确退款条件
退款条件
Fly 提供沙盒验证（免费）和企业版两种服务。企业版客户在以下条件下可申请退款：
退款申请流程
① 登录 Dashboard → 设置 → 订阅管理 → 申请退款 → ② 填写退款原因和退款金额 → ③ 1-3 个工作日审批 → ④ 审批通过后 3-5 工作日到账。如有疑问，联系 support@fly-agent.xyz。
不可退款情况
以下情况不支持退款：① 免费沙盒环境（本身免费）；② 签约超过 30 天且已使用超过一个月的企业版；③ 因客户自身原因（如 API Key 泄露）导致的服务中断。
