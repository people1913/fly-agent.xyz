隐私政策

Fly — Commercial Trust Protocol

最后更新：2026-07-07

1. 概述

Fly（"我们"、"本协议"）是 Agent Economy 的商业信任协议。本隐私政策说明我们如何收集、使用、存储和保护通过 Fly 协议处理的数据。Fly 的核心设计原则是最小数据收集——我们只处理验证商业信任所必需的数据。

2. 我们收集的数据

2.1 协议层数据

Fly 协议处理以下类型的数据：

  • Claim（声明）：参与方身份、交易类型、时间戳
  • Evidence（证据）：行为记录、数据来源、哈希值
  • Rule（规则）：归因规则定义、发行方信息
  • Trust Record（信任记录）：归因结果、结算建议

以上数据均为商业事实数据，不包含个人身份信息（PII），除非参与方主动提供。

2.2 网站访问数据

当您访问 fly-agent.xyz 时，我们可能收集：

  • IP 地址（匿名化）
  • 浏览器类型和版本
  • 访问时间和页面
  • 引荐来源

3. 数据使用

我们使用收集的数据用于：

  • 生成和验证 Trust Record
  • 执行 Deterministic Replay 验证
  • 改进协议实现
  • 网站性能分析

我们不会将协议数据用于广告投放、用户画像或第三方数据交易。

4. 数据存储与保护

  • 协议数据通过 SHA-256 哈希保护，确保证据不可篡改
  • Trust Record 一经生成不可修改，更新必须生成新版本
  • 网站数据存储在加密环境中
  • 我们定期审查安全措施

5. 数据共享

Fly 协议的核心价值在于可验证性。Trust Record 可被任何独立方验证，但这不意味着数据被"共享"——验证方只能看到他们已有权限访问的数据。

我们不会向第三方出售或出租任何数据。

6. 您的权利

根据适用法律，您可能有权：

  • 访问我们持有的关于您的数据
  • 请求更正不准确的数据
  • 请求删除您的数据（在法律允许范围内）
  • 反对或限制数据处理
  • 数据可携权

如需行使上述权利，请通过 privacy@fly-agent.xyz 联系我们。

7. Cookie 政策

fly-agent.xyz 使用必要的 Cookie 以确保网站正常运行。我们不使用广告 Cookie 或追踪 Cookie。

8. 第三方服务

我们使用以下第三方服务：

  • Cloudflare：CDN 和安全保护
  • GitHub Pages：静态网站托管

这些服务有其各自的隐私政策，我们建议您查阅。

9. 政策更新

我们可能不时更新本隐私政策。重大变更将通过网站公告通知。继续使用 Fly 即表示您同意更新后的政策。
