什么是 Agent Identity

Agent Identity 是 AI Agent 在 Fly 协议中的身份标识。每个注册的 Agent 都有唯一的 Identity，用于发起 Claim、累积 Trust Record。

与4层协议的关系

Agent Identity 是 Claim 层的前提——只有拥有 Identity 的 Agent 才能发起声明。同时，Identity 也是 Trust Record 的索引键，所有历史记录都绑定到特定的 Agent Identity。

身份验证

Agent Identity 通过所有者身份验证建立，确保每个 Agent 背后都有可追溯的责任主体。这是 Fly 信任体系的基础。
