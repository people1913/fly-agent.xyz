Base 归因的独特价值
  • Coinbase 拥有 1 亿+ 用户，但大部分用户行为无法被外部归因追踪
  • Base 作为 L2，Gas 极低（<$0.01），可以高频归因而不担心成本
  • Coinbase Wallet + Onchain Kit 提供了原生的链上身份入口
  • Base 生态快速增长，SocialFi（Farcaster）和 Creator Economy 场景丰富
  • Coinbase 用户从法币入金，是"Web2 → Web3"的关键桥梁

支持场景

Coinbase 用户归因
AI Agent 推荐了 Base 上的项目 ↓ 用户从 Coinbase 直接桥接到 Base ↓ 用户在 Base DApp 上交互 ↓ Fly 归因: "这个用户从 Coinbase 入金，来自 AI 推荐"
SocialFi 归因
AI Agent 推荐了 Farcaster 上的内容 ↓ 用户在 Base 上 mint 相关 NFT ↓ 用户打赏/购买社交代币 ↓ Fly 归因: "这笔社交互动来自 AI 推荐"
Creator Economy 归因
AI Agent 推荐了某创作者的订阅 ↓ 用户在 Base 上购买创作者 NFT/代币 ↓ Fly 归因: "这个订阅来自 AI 推荐" ↓ 创作者可以向 AI Agent 支付佣金
高频小额归因
Base Gas <$0.01，支持： ↓ 社交点赞 → 归因 ↓ 评论打赏 → 归因 ↓ 小额 NFT 购买 → 归因 ↓ 在 Ethereum 上成本太高的场景，Base 可以做

Coinbase 集成

// 使用 Onchain Kit 集成 Coinbase Wallet
import { useCoinbaseWallet } from '@coinbase/onchainkit/wallet';

// 连接钱包时，同步注册到 Fly
const { connect } = useCoinbaseWallet();
const { address } = await connect();

// 注册到 Fly
await flySDK.registerWallet({
  chain: 'base',
  address: address,
  provider: 'coinbase_wallet'
});

// 后续交易自动归因
const tx = await contract.swap(...);
await flySDK.attributeTransaction({
  tx_hash: tx.hash,
  chain: 'base'
});
