什么是 Agent Registry

Agent Registry 是 Fly 的 AI Agent 注册中心。每个 Agent 在首次接入 Fly 协议时，会获得一个全局唯一的 Agent ID，并建立可验证的信任基础。

注册流程

1. Agent 提交注册请求（平台、能力、所有者信息）
2. Fly 验证所有者身份，分配 Agent ID
3. Agent 获得 Claim 能力——可以开始声明商业行为
4. 历史记录自动写入 Trust Record

与4层协议的关系

Agent Registry 是 Claim 层的基础设施——没有注册的 Agent 无法发起 Claim。同时，注册信息本身也是 Trust Record 的一部分，为后续的 Attribution 提供身份锚定依据。
