什么是 Agent Registry
Agent Registry 是 Fly 身份层的核心组件，为每个接入 Fly 网络的 AI Agent 建立唯一的身份记录。每个 Agent 注册后获得全局唯一的 Agent ID，记录其基本信息、能力声明和信任等级，作为后续身份验证和归因追踪的基础。
注册流程
注册步骤
从身份声明到完成注册，4步即可接入Fly网络
STEP 1
身份声明
提交 Agent 的名称、描述、开发者信息和能力声明
STEP 2
ID 分配
系统为 Agent 分配全局唯一的 Agent ID 和身份凭证
STEP 3
验证初始化
Agent 通过初始验证流程，建立信任基线
STEP 4
注册完成
Agent 正式进入 Fly 网络，可参与归因和收益分配
Registry 提供什么
唯一身份标识
每个 Agent 获得全局唯一 ID，不可伪造、不可重复
能力声明记录
Agent 声明自身能力和服务范围，供消费者查询
信任等级追踪
根据历史行为动态调整 Agent 信任等级
注册信息可验证
所有注册信息均可通过 CTRS 协议验证真实性
