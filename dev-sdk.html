TypeScript / Python SDK，3行代码开始归因
为什么使用 SDK
直接调用 API 需要处理认证、重试、错误处理等逻辑。Fly SDK 封装了所有底层细节，3行代码即可开始归因验证。
快速开始
TypeScript: import { Fly } from "@fly/sdk"; const fly = new Fly({ apiKey: "..." }); await fly.action.create({ source: "douyin", referrer: "agent_008" });
Python: from fly import Fly; fly = Fly(api_key="..."); fly.action.create(source="douyin", referrer="agent_008")

TypeScript SDK
npm install @fly/sdk — 完整类型定义，IDE 自动补全，支持 Node.js 和浏览器端
Python SDK

自动 Behavior ID
SDK 自动为每个 AI 推荐生成 Behavior ID，无需手动配置，开箱即用

fly dev 命令启动本地调试服务器，实时查看归因数据、模拟验证流程
